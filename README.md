---
title: Obtaining the Text 
format: html
---
1. Install R and RStudio according to [Getting Started With R](https://noah.ees.hokudai.ac.jp/yoshida/html/Genetics/2026/GettingStartedWithR.html)

2. Access Yoshida's github site

  [Yoshida's github](https://github.com/kiyohitoyoshida/Genetics)

3. Download the ZIP archive file from the "Code" icon.

  [Code](https://github.com/kiyohitoyoshida/Genetics/archive/refs/heads/main.zip)

4. Copy the archive file and paste it under your "Documents" folder.
5. Unzip it by right-clicking and selecting "Extract All" from the menu.
6. Start RStudio.
7. File > New Project > Existing Directory
8. Select "text" folder under the archive folder "Genetics-main".
9. "Create Project"
10. Open "Drosophila.qmd" file.
11. Click the "Render" menu.
12. You can see the text in your browser or open "Drosophila.html" file.
13. Open "Gantt.qmd" file and "Render".
14. You can modify the schedule by editing the Gantt.qmd file.
15. Close the project and quit RStudio.
